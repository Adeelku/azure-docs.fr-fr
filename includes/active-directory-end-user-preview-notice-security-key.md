> L’utilisation d’une clé de sécurité comme méthode d’authentification non basée sur un mot de passe est actuellement en version préliminaire publique. Si ce que vous voyez sur votre écran ne correspond pas à ce qui est qui est abordé dans cet article, cela signifie que votre administrateur n’a pas encore activé cette fonctionnalité. Tant que cette fonctionnalité n’est pas activée, vous devez choisir une autre méthode d’authentification sur la page [**Informations de sécurité**](https://docs.microsoft.com/azure/active-directory/user-help/user-help-security-info-overview). Pour plus d’informations sur les préversions, consultez [Conditions d’utilisation supplémentaires pour les préversions de Microsoft Azure](https://azure.microsoft.com/support/legal/preview-supplemental-terms/).
